# Drum Machine

This project is a web-based drum machine built as part of the freeCodeCamp "Front End Development Libraries" curriculum.

[![Drum Machine Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://kielakjr.github.io/drum-machine)

## Description

This interactive drum machine allows users to play sounds by clicking on drum pads or by using corresponding keyboard keys. It's built with modern web technologies, providing a responsive and engaging user experience.

## Features

*   **Interactive Drum Pads:** Click on any of the drum pads to trigger a sound.
*   **Keyboard Support:** Use your keyboard to play the drums. Each pad is mapped to a specific key.
*   **Responsive Design:** The layout is optimized for a seamless experience on both desktop and mobile devices.

## Technologies Used

This project is built with:

*   **[React](https://reactjs.org/)**: A JavaScript library for building user interfaces.
*   **[Vite](https://vitejs.dev/)**: A modern frontend build tool that provides a faster and leaner development experience.
*   **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.

## Setup and Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine.

### Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/kielakjr/drum-machine.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd drum-machine
    ```
3.  Install NPM packages:
    ```sh
    npm install
    ```

### Available Scripts

*   To run the app in development mode:
    ```sh
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

*   To build the app for production:
    ```sh
    npm run build
    ```

*   To lint the code for issues:
    ```sh
    npm run lint
    ```
